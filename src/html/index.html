Pasala!

This game is for passing the ball between players. You can win counting the number of passes or by scoring a goal.

Play!

Instructions

  1. Start passing the ball with the Goal Keeper. You can passing long or short.
  2. You can pass the ball to any player on the field.
  3. You are the blacks.
  4. Select a player to pass the ball.
  1. You have two moves per turn.
  2. First move one player, then pass the ball to anybody. You can choose the player you already moved or another.
