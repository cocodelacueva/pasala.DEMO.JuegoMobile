Pasala!

This game is for passing the ball between players. You can win counting the number of passes or by scoring a goal.

Play!

Points: 0

Goals: 0

Dificulty: 1

Instructions

  1. Start passing the ball with the Goal Keeper. You can passing long or short.
  2. To pass, click the player who has the ball, then click the player you want to pass the ball to.
  3. You are the blacks.
  4. If you want to move a player, before select the ball, select a player instead.
  5. The player selected will become orange
  6. Click where do you want to move, the player will not move too long
The ball was intercepted. Game is over!
